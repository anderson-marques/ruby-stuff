# Three ways to use Ruby

- Single command

- Ruby file

- Interactive Ruby Shell (IRB)
  - Allows us to interact with code in real time
  - Great for testing code

```bash
host@user:~$ irb
irb(main):001:0> puts 123
123
=> nil
irb(main):002:0>quit
host@user:~$ irb --simple-prompt
>>
```
