# Floats (precisition)

```ruby
  12.3012 => 12.3012
  12.3012.class => Float
  x = 10 => 10
  y = 10.0 => 10.0
  x.class => Fixnum
  y.class => Float
  # Integer divided by integer return integer
  10 / 3 => 3

  # Float divided by integer or the opposite returns float
  10.0 / 3 => 3.33333...
  10 / 3.0 => 3.33333...

  # Round method convert the number to an intenger
  12.678.round => 13

  # The to_i method converts to integer explicitly
  12.678.to_i => 12
```
